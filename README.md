A Python-based scheduling simulation project using **SimPy**. The project focuses on task scheduling, priority index calculation, and performance analysis.

Key points:

- Implemented scheduling simulation using Python and SimPy
- Computed Priority Index (PI)
- Visualized results using logs and Gantt chart
- Compared performance metrics such as waiting time and deadline tardiness

## How to Run Locally

To run this website locally:

1. Clone this repository.

```bash
git clone https://github.com/Milahsaa/Portfolio.git
```

2. Open the project folder.

```bash
cd Portfolio
```

3. Open the HTML file in a browser.

```text
index.html
```

or:

```text
main.html
```

---

## GitHub Pages Deployment

This portfolio is deployed using **GitHub Pages** from the `main` branch.

To make the website open directly from the GitHub Pages link, the main HTML file should be named:

```text
index.html
```
